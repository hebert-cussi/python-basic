## Descripción

Curso intensivo de programación que introduce a los participantes en Python, el lenguaje más popular del mundo para ciencia de datos, automatización y desarrollo web. Sin conocimientos previos requeridos, el estudiante progresa desde los conceptos más básicos hasta construir un proyecto integrador funcional en solo dos semanas.

## Objetivos de aprendizaje

Al finalizar el curso, el participante será capaz de:

- Escribir y ejecutar programas en Python usando variables, tipos de datos y operadores.
- Controlar el flujo de un programa mediante condicionales y bucles.
- Definir y utilizar funciones para organizar y reutilizar código.
- Manejar las principales estructuras de datos: listas, tuplas, diccionarios y conjuntos.
- Aplicar manejo de errores y leer/escribir archivos de texto.
- Desarrollar un mini sistema de registro como proyecto final integrador.

## Requisitos y recursos

**Pre-requisitos:**
- Manejo básico de computadora y navegador web.
- No se requieren conocimientos previos de programación.

**Recursos necesarios:**
- Computadora personal con acceso a internet.
- Editor de código: Visual Studio Code (descarga gratuita).
- Python 3.10 o superior instalado (se guiará en la sesión 1).
- Navegador web: Google Chrome o Mozilla Firefox (última versión).

## Proyecto final

Cada participante desarrollará un **mini sistema de registro de estudiantes** que integre los conocimientos del curso, presentado en la última sesión.

**Entregables mínimos:**
- Programa funcional en Python que permita registrar, listar y buscar estudiantes.
- Uso de al menos una función propia, un diccionario y una lista.
- Manejo de errores con `try/except` para entradas inválidas del usuario.
- Guardado y lectura de datos desde un archivo de texto (`.txt`).
- Código limpio con comentarios y docstrings en las funciones.
- Demo en vivo + presentación breve de 5 minutos ante el grupo.
